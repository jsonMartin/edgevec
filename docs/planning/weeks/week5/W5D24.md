# W5D24 - Advanced Fuzzing

## Goal
Stress-test the persistence layer and graph invariants under chaotic conditions.

## Specifications
- **Fuzz Targets:**
  1. `fuzz_persistence`: Generate random byte streams -> Attempt `load`.
  2. `fuzz_graph_ops`: Random sequence of `Insert` / `Delete` / `Search`.

## Tasks
1. [ ] Create `fuzz/fuzz_targets/persistence_load.rs`.
2. [ ] Create `fuzz/fuzz_targets/graph_ops.rs`.
3. [ ] Run fuzzer for 1 hour.
4. [ ] Fix any panics discovered.

## Acceptance
- No panics (crashes) allowed.
- Memory leaks detected by sanitizer are fixed.
